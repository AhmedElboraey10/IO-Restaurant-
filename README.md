# IO Restaurant | The Finest Cuisine Landing Page

## Project Overview
This project is a professional, high-end landing page for **IO**, a specialized seafood restaurant. The primary goal was to create an immersive digital experience that reflects the quality, freshness, and atmosphere of the restaurant.

** Live Demo:** [Experience IO Restaurant Here](https://ahmedelboraey10.github.io/IO-Restaurant-/)

What makes this project unique is its **evolutionary path**. It started as a foundational exercise based on a well-known creative agency template (**Elzero Template Two / Kasper**) and was systematically re-engineered, styled, and repurposed to serve the hospitality and fine-dining sector.

---

## The Transformation (From Template to Brand)
The core challenge was to break away from the "generic agency" look and feel. Every section was audited and modified to serve a new logic:
- **Concept Rebranding:** Shifted the identity from a "Creative Agency" to a "Premium Seafood Restaurant."
- **Logic Adaptation:** Converted subscription plans into **Set Menus**, and general services into **Restaurant Value Props** (Fresh Ingredients, Chef Expertise, etc.).
- **Dynamic Content:** Replaced standard text with engaging hospitality-focused copy that emphasizes flavor and quality.
- **Visual Overhaul:** A complete departure from the original color palette, moving to a warm, appetite-stimulating theme (`#ff8c00`) with custom-optimized assets.

---

## Key Features

- **Immersive Hero Section:** A full-screen landing area featuring a background video overlay and a smooth navigation system.
- **Tailored Services & Value Props:** Four core pillars of the restaurant’s service are highlighted with custom iconography and descriptive text.
- **Chef’s Recommendations:** A specialized section showcasing signature dishes with a focus on UI details like custom borders and active states.
- **Categorized Gallery:** An interactive gallery with a filtering system to browse through Appetizers, Main Courses, Desserts, and Atmosphere.
- **Advanced Reservation System:** A tailored booking form that includes guest count validation, date/time selection, and special requests.
- **Social Proof & Statistics:** Integrated sections for real-world metrics and customer testimonials.

---

## Technical Highlights
- **Semantic HTML5:** Using meaningful tags for better SEO and accessibility.
- **Modular CSS3:** Built with a focus on maintainability, utilizing CSS Variables for global styling.
- **Responsive Architecture:** The layout is fully fluid, using a mix of Flexbox, CSS Grid, and the `max()` function for dynamic scaling across all device sizes.
- **Performance Optimization:** Usage of `.webp` format for all images and optimized video backgrounds to ensure high quality with low file sizes.
